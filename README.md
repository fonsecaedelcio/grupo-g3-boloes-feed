# Grupo G3 — vitrines de bolões

Atualização automática dos bolões disponíveis nas três unidades do Grupo G3 Lotéricas.

- Casa Lotérica WE — código 2484
- Praça XI — código 20665
- Lotérica Tomazini — código 13979

O coletor consulta a fonte pública da CAIXA, publica somente ofertas válidas e preserva a última vitrine correta quando a fonte estiver temporariamente indisponível ou responder vazia.

A compra é sempre concluída no ambiente oficial da CAIXA. Serviço destinado a maiores de 18 anos.
